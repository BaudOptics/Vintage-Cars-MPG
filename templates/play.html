Machine Learning Analysis

Introduction


In this project we used various machine learning models, trained and tested the data to see the models ability to predict the MPG (Mile Per Gallon) for a vehicle and tells us about the efficiency of fuel consumption of a vehicle in the 70s and 80s base on other attributes of that vehicle. We used linear, decision tree and random forest models.

Data Preprocessing


The first step in our analysis was to clean and pre-process our dataset to make ready for our machine learning analysis. The dataset was imported into Jupyter Notebook and read into a pandas dataframe. Data was examined for null values and understood prior to machine learning model implementation.



To enhance our understanding of the data distribution, we looked at the histogram of numerical features.


It was observed that "origin" and "cylinders" are far from normal distribution and for that reason this features were analyzed and label encoded as categorical.


Correlation


The gerenarion of a pair plot was able to give us a brief overview of how each variable behaves with respect to every other variable. In this case, the MPG column (the target variable) is negatively correlated with the displacement, weight, and horsepower features.

Also, the pair plot clearly demonstrates that "cylinders" and "origin" fields do not shows a normal distribution as they represent a specific value and can be considered categorical values.


In the correlation analysis we have found that weight and displacement have the highest correlation with MPG. On the other hand, origin and number of cilynders have the lowest correlation.


Selecting and Training Machine Learning Models


Since this is a regression problem, the following models were selected:

1. Linear Regression
2. Ridge
3. Lasso
4. ElasticNet
5. Decision Tree Regressor
6. Random Forest Regressor
7. Ada Boost Regressor
8. Gradient Boosting Regressor

Each of the models were evaluated using Root Mean Squared Error (typical performance metric for regression problems).



Based on this comparison, the Random Forest model was choosen because it has the lowest RMSE and doesn't overfit the in sample data.
